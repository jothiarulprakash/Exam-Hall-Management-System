About

Hall Allotment for invigilators during any examination is done manually right now which involves manual intervention each and every time.
Paper work is done to assign halls for invigilators. Is there any web application to automate this process? The answer was a NO.
Then, Why not we create it? We are purely aiming at developing the software to automate hall allotment for invigilators during exams specifically for our college.

On The Implementation Perspective,
Database ( MySQL 5.6.11 ) is chosen as Back end and Web Technology ( HTML5, CSS3 ) is chosen as the Front end and the processing is looked after by ( PHP 5.4.14 ).
All these are done on EasyPHP 13.1 server (Apache server 2.4.4 for Windows) and the recommended browser being Firefox 30.0(and above). Interface will be much simpler and easier to interact with. This software once developed will cut manual labour by about say 50% for hall allotment during exams.
